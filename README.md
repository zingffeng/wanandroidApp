# WanAndroid客户端（Java语言）

## 简介

WanAndroid项目采用Java语言编写，使用Retrofit、Glide、ARouter、Butterknife、SmartRefreshLayout等流行第三方框架搭建的应用程序

## 实现功能

- 首页、广场、体系、项目四大模块；
- 登录注册功能；
- 搜索功能：热门搜索、搜索历史；
- 浏览文章、用浏览器打开文章；
- 我的积分
